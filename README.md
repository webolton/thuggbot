# Thuggbot

## A program that takes pictures of my cats and tweets them to me

## Prereqirements

- python 3.9.0 >=
- pip3
- pytest

## Install the dependencies

`make setup`

## Run the project

`make run`

## Run the tests

`make test`

## Run the linter

`make lint`

## Export requirements.txt

`make reqs`
