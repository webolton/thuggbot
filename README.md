# Thuggbot

## A program that takes pictures of my cats and tweets them to me

## Prereqirements

- python 3.9.0 >=
- pip3

## Install the dependencies

`make setup`

## Run the project

Requires one of two arguments: `cat` for cat recognition and identification, `human` for humans.

`make run type=cat`

## Run the tests

`make specs`

## Run the linter

`make lint`

## Export requirements.txt

`make reqs`
